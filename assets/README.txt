MeTriqo — Image assets
======================

Drop your product photos in this folder with these exact names:

  mixer.png     →  the Scarlett Super Hand Mixer photo
  grinder.png   →  the Electric Grinder photo
  combo.png     →  (optional) a combined mixer+grinder photo

Tips:
- PNG with transparent background looks best on the colored tiles.
- Square-ish images (e.g. 800x800) fit the cards perfectly.
- If an image is missing, the page shows a striped placeholder with the
  filename, so the layout never breaks.

The MeTriqo logo is drawn as inline SVG (no file needed). If you prefer
to use the real logo image, replace the <svg> in the header of index.html
with: <img src="assets/logo.png" alt="MeTriqo" width="40">
